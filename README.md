## What is this？
由於今年Covid19在台灣本土爆發，導致許多店家開始使用實名制的方式進行相關疫調工作。
但發現各地都產生許多實名制的亂象，為了改變並且簡化實名制，民眾不需要再輸入過多的個資。
於是我著手利用空閑時間進行開發這套經過簡訊實名制的系統。
